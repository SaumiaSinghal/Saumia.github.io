During my Summer Research Internship, I worked on the topic "A Microscopic Traffic Simulator Integrated Driving Simulator". We developed a server using sockets for the real-time exchange of data between a traffic simulator, VISSIM and a gaming engine (Unity3D), which functioned as a Driving Simulator. We furthur provided the user the control of a simulator vehicle and sent its trajectories back to VIssim so that other vehicles, present in the scenario, can change their behaviours on the basis of user behaviour. During this project, I got a chance to work on Virtual Reality principles as this whole project was transferred from Unity to Oculus Rift, for better user experience.

We are furthur planning to transfer this project to big screens while taking user input from controllers.
